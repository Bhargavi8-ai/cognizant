CSS3 Advanced Features

Master modern CSS3 capabilities for responsive and interactive web design

1. CSS3 Transitions

Smooth animations between CSS property changes over time.

Hover Me!

.box-transition {
    background-color: #007BFF;
    transition: all 0.3s ease;
}

.box-transition:hover {
    background-color: #ff6b6b;
    transform: scale(1.1);
}

3. CSS3 Animations

Create keyframe animations for complex movement sequences.

Animated Box
🔴
Fading In

@keyframes slideIn {
    from {
        transform: translateX(-100%);
        opacity: 0;
    }
    to {
        transform: translateX(0);
        opacity: 1;
    }
}

.box-animate {
    animation: slideIn 2s ease-in-out;
}

9. Pseudo-elements (::before, ::after)

Add decorative content without modifying HTML.

Hover for tooltip
This is a blockquote with styled pseudo-element

/* Using ::before */
.quote::before {
    content: '"';
    font-size: 3em;
    color: #007BFF;
}

/* Using ::after */
.btn-with-icon::after {
    content: '→';
    margin-left: 10px;
}

/* Tooltip */
.tooltip::after {
    content: 'Tooltip text';
    position: absolute;
    background: #333;
    color: white;
    padding: 5px 10px;
}

11. CSS3 Media Queries

Responsive design with advanced media query features.

Resize your screen to see changes

/* Screen Width */
@media (max-width: 768px) {
    body { font-size: 14px; }
}

/* Screen Orientation */
@media (orientation: landscape) {
    .sidebar { display: none; }
}

/* Device Features */
@media (prefers-reduced-motion: reduce) {
    * { animation: none !important; }
}

/* Color Scheme */
@media (prefers-color-scheme: dark) {
    body { background: #222; color: #fff; }
}

12. CSS3 Custom Properties (CSS Variables)

Define and use reusable CSS variables.

Light Theme
Dark Theme

/* Define Variables */
:root {
    --primary-color: #007BFF;
    --secondary-color: #ff6b6b;
    --padding: 20px;
    --transition: all 0.3s ease;
}

/* Use Variables */
.button {
    background-color: var(--primary-color);
    padding: var(--padding);
    transition: var(--transition);
}

/* Override in Specific Context */
.dark-theme {
    --primary-color: #333;
    --secondary-color: #fff;
}
